# Smart Home Automation System

## Overview
This project presents an Arduino-based Smart Home Automation System that uses sensors and actuators to automate household appliances.

## Features
- Automatic lighting control
- Motion detection
- Obstacle detection
- Distance measurement
- Motor control using L293D
- Sensor-based automation

## Components Used
- Arduino Uno
- PIR Sensor
- LDR Sensor
- IR Sensor
- Ultrasonic Sensor
- Servo Motor
- DC Motor
- L293D Motor Driver

## Technologies
- Arduino IDE
- Embedded C
- Electronics
